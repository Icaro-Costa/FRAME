# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão mais recente deste projeto recebe atualizações de segurança.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Levamos a segurança deste projeto a sério. Se você encontrar alguma vulnerabilidade de segurança, por favor, **NÃO** a divulgue publicamente imediatamente.

Siga os passos abaixo:

1.  Envie um e-mail para [icaro.developerr@gmail.com](mailto:icaro.developerr@gmail.com).
2.  No assunto, coloque: `[ALERTA DE SEGURANÇA] - FRAME`.
3.  Descreva detalhadamente a vulnerabilidade encontrada e, se possível, os passos para reproduzi-la.

Nossa equipe (Grupo Arquimedes) analisará o relatório e responderá o mais breve possível. Agradecemos sua colaboração em manter o projeto seguro para todos.
